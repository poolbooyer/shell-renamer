# about this
このリポジトリはファイル名置換処理を行うスクリプトをshellとGoogle Apps Scriptで作成し、比較するためのものです。
## In this repository
### generator
- generator.py
  - ベースになるhtmlファイルを連番で100個性生成するのに使用しています。
## shell
- renamer.sh
  - ファイル名称を置換するシェルコードです。
  - generatorで生成される連番のファイルを連番のまま名称を変更する様に作成してあります。
